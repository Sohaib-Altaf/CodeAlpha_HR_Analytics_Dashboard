# HR Analytics Dashboard — CodeAlpha Task 2

## Overview
Interactive HR Analytics Dashboard built using Power BI 
with real IBM HR Dataset (1,470 employees)

## Dashboard Pages
- Page 1: Overview KPIs
- Page 2: Attrition Analysis  
- Page 3: Satisfaction & Performance
- Page 4: Predictive Forecast

## Key Insights
- Overall Attrition Rate: 16.1%
- Age 18-25 highest risk: 34.8%
- Sales department: 20.6% attrition
- Potential cost saving: $218,400/year

## Tools Used
- Power BI Desktop
- DAX Formulas
- IBM HR Dataset (Kaggle)
